# nlw_nodejs
 
### Comandos de inicialização de projeto node

    npm init -y

adicionador Typescript:

    npm add typescript -D

inicializar Typescript:

  Linux

    npx tsc --init
  
  Windows
  
    npm tsc --init

converter o código Typescript para JavaScript:

  Linux

    npx tsc

  Windows

    npm tsc

### Adiciona o Express

    npm add express
